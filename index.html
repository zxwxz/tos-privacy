AutoPost2025

A lightweight personal tool that uploads and schedules 720 p AI‑generated short videos to a TikTok account automatically, via the official TikTok Content Posting API.

How It Works

  1. Generate a 5‑second 720 p clip on mobile (WAN 2.1 / Veo etc.).
  2. The script calls /v2/post/publish/video/init with source=FILE_UPLOAD.
  3. TikTok publishes the video at the specified publish_time.

Data Usage & Privacy

The tool stores only the OAuth access token and video filenames required for uploading. No personal data is collected or shared.
